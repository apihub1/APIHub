Perguntas Frequentes

Como começar a usar uma API do catálogo?

Para começar a usar uma API do nosso catálogo, primeiro localize a API desejada na página de catálogo. Clique no botão "Copiar" para obter o endpoint da API. Veja na descrição da API como usar ela. Coloque em seu sistema/site/software e bom proveito!

Preciso pagar para usar as APIs?

O APIHub é um catálogo que reúne informações sobre APIs disponíveis publicamente. Algumas APIs são gratuitas, enquanto outras são pagas, isso varia dependendo da complexidade.

Por que algumas APIs são tão lentas?

Infelizmente, o APIHub ainda não tem nenhum lucro monetário, nenhum investimento aplicado e nenhum dinheiro arrecadado. Por isso, algumas de nossas APIs são hospedadas em serviços gratuitos, tornando-as mais lentas por não ficarem sempre ativas, tendo sempre que iniciar o servidor após um tempo de inatividade.

Tenho uma API no APIHub, como receber reconhecimento/remover?

Primeiramente perdão, pois nós da APIHub muitas vezes achamos as APIs open, mas não seus criadores.
Mas para receber o seu crédito ou solicitar a remoção dela é bem simples, apenas nos envie um e-mail solicitando o reconhecimento ou a remoção da API. Juntamente com o e-mail, também nos envie provas de que tal API é sua ou foi criada por você. E seu caso estará resolvido!

Como resolver erros comuns de autenticação?

Erros de autenticação geralmente ocorrem por esses motivos:

  • Chave de API ausente ou inválida
  • Permissões insuficientes para a operação solicitada
  • IP não autorizado (quando há restrições por endereço IP)

Verifique se você está incluindo a chave de API no cabeçalho ou parâmetro correto.
